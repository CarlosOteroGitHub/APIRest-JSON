========================================================== COMANDOS ==========================================================

° npm init -y: Comando utilizado para iniciar un proyecto NodeJS.

° npm install express: Comando utilizado para instalar la última versión de Express.

° npm install path: Comando utilizado para instalar la última versión de path.

° npm install dotenv: Comando utilizado para instalar la última versión de dotenv.

° npm install ejs: Comando utilizado para instalar la última versión del software de plantillas Easy Java Simulations.

° npm i nodemon -g: Comando utilizado para instalar la última versión del servidor nodemon.

° nodemon [nombre_archivo]: Comando utilizado para ejecutar un archivo Javascript en el servidor. El servidor corre por defecto en el navegador con la dirección localhost:8080

    NOTA: Para consultar los datos JSON realice una petición HTTP GET a http://localhost:8080/api/users
